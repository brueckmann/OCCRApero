# SERIOUS GAMES IN ENERGY POLICY
A short presentation (https://brueckmann.github.io/OCCRApero/) including the [SWEET SURE](sweet-sure.ch) logo prepared by G. Brückmann for the [Oeschger Centre for Climate Change Research (OCCR)](https://www.oeschger.unibe.ch/) Apéro on 23 September 2025.

## Information

**Purpose** 
All code for generating and publishing the presentation on serious games are at the same location.  

**Author &** [Contact](brueckmann.github.io "Contact info on my Website") 

G. Brückmann, University of Bern 

**Prerequisites**
- R (was created using R version 4.4.0), see also `session-info.tex`  
- RStudio was used to open the Rproject 


**Folder structure**
- `assets/` contains inputs for the design  
- `docs/` contains outputs including the website  
- `session-info.tex`
